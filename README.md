## ✨ About

**E-Commerce Portfolio** is a clean, responsive, and modern personal website built for an e-commerce professional.

It presents the profile, skills, services, and experience in a professional and elegant way.

---

## 🎨 Features

- Clean and modern UI
- Fully responsive design
- Professional personal branding
- Skills and services section
- Smooth user experience
- Built with modern web technologies

---

## 🛠️ Tech Stack

| Technology | Usage |
| ---------- | ----- |
| **Next.js** | Website framework |
| **React** | UI components |
| **Tailwind CSS** | Styling |
| **HTML / CSS** | Structure and design |

---

## 🚀 Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the project:

```bash
http://localhost:3000
```
